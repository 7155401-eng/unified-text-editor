## 2026-05-17 - Missing ARIA Labels on Icon Buttons\n**Learning:** Many icon-only buttons (like AI toggles and format actions) lacked `aria-label` attributes despite having `title`s. While `title` provides tooltip hover info, `aria-label` provides robust accessibility for screen readers on icon buttons.\n**Action:** Always verify icon-only interactive elements possess explicit `aria-label`s for accessibility.
