# דמו TipTap — שלב 5

דמו עברי בסיסי לוודא ש‑TipTap עובד עם:
- עברית
- ניקוד
- טעמי מקרא
- כיוון RTL
- עיצוב טקסט (מודגש, נטוי, כותרות, רשימות, יישור)

## איך לצפות

### דרך 1 — שרת מקומי (כבר רץ)
פתח דפדפן: <http://127.0.0.1:5173/>

### דרך 2 — להפעיל ידנית
```
cd C:\Users\User\arch_work_remote\engine-into-editor\tiptap_sandbox
npm run preview
```
לאחר מכן פתח <http://127.0.0.1:5173/>.

### דרך 3 — בנייה לקובץ סטטי
```
npm run build
```
אחר כך פתח את `dist\index.html` בדפדפן.

## מה לבדוק

1. הקלד טקסט עברי כלשהו.
2. לחץ "טען טקסט עברי לדוגמה" — מופיע טקסט עם ניקוד וטעמים.
3. בדוק שכל סימני האבחון בצד ימין מסומנים `✓`.
4. נסה כפתורי עיצוב: מודגש, כותרות, רשימות, יישור.
5. RTL ↔ LTR מחליף כיוון.
6. "ייצא JSON" — מוצג מבנה הנתונים של TipTap בקונסולה ובחלון.

## אבחון אוטומטי

חלון האבחון מציג בזמן אמת:
- האם העורך נטען
- האם RTL פעיל
- האם עברית מזוהה (תווים בטווח Unicode עברי)
- האם ניקוד מזוהה
- האם טעמי מקרא מזוהים
- האם פלט JSON תקין

## עצירה

- שרת רץ בחלון הטרמינל. לסיום: `Ctrl+C`.
- אם נסגר ולא רץ — הפעל מחדש עם `npm run preview`.
